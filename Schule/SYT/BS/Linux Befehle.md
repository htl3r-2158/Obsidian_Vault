_Liste aller Linux Befehle_
- ```sudo``` … Super User
- ```cp``` … Kopieren
- ```pwd``` … Print working directory
- ```ls``` … list directory
- ```man``` … Handbuch
- ```mv``` … move
- ```mkdir``` … make directory
- ```cat``` … gibt datei aus
- ```rm``` … remove
- ```tree``` … Baumdarstellung directory
- `history` … zeigt alle letzten Befehle an
