Begründe warum die (u+1)te Ableitung von Grad "$n$" gleich Null ist:
a) $y=x^5$
		$y'=5x^4$
	$y''=20x^3$
	$y'''=60x^2$
	$y'''' = 120x$
	$n: \ y^{5} = 120$
	$n+1: y^{6} = 0$
b)
	$y=3x^4 + 2x^3 - x + 5$
	$y' = 12x^3 + 6x^2 -1$
	$y'' = 36x^2 +12x$
	$y''' = 72x + 12$
	$n: \ y^{(4)} = 72$
	$n+1: \ y^{(5)} = 0$
