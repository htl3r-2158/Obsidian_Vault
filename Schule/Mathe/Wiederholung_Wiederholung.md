# Produktregel - Partielle Integration

$[u(x) * v'(x)]' = u'(x) * v(x) + u(x)*v'(x) -> Integretation -> u(x)*v(X) = \int u'v * dx + \int uv' * dx$

> $\int u*v' dx = u*v - \int u'*v dx$

oder.: $\int u'*v dx = u*v - \int u*v' dx$
